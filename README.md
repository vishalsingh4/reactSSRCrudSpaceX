# reactSSRCrudSpaceX
A react redux application with server side rendering for the SpaceX launches with year, launch success and land success filters. 
Jest and enzyme used for writing unit test cases.

Install Dependencies
 - npm i
 
 Run locally
 - npm run start-dev
 
 Test
 - npm run test
 - npm run test:watch
 
 Travis
 - CI/ CD process
